# Restaurant Mapping

web ui submission dicoding katalog restoran 

# Sumber:

https://myanimelist.net

# Installasi

- Jalankan perintah di terminal

```sh
# clone repo
git clone https://github.com/shoya-m1/AnimeList.git

# masuk folder
cd AnimeList

# install dependensi
npm install

# jalankan server
npm run serve
```
