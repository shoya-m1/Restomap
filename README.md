# Restorant Mapping

Web ui submission dicoding kelas expert katalog restoran 

# Installasi

- Jalankan perintah di terminal

```sh
# clone repo
git clone https://github.com/shoya-m1/AnimeList.git

# masuk folder
cd Restomap

# install dependensi
npm install

# jalankan server
npm run start-dev
```
