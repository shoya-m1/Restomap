# Restaurant Mapping

web ui submission dicoding katalog restoran 

# Installasi

- Jalankan perintah di terminal

```sh
# clone repo
git clone https://github.com/shoya-m1/Restomap.git

# masuk folder
cd Restomap

# install dependensi
npm install

# jalankan server
npm run start-dev
```

- Demo Website https://restomap.vercel.app/
